# Security Policy

## Supported Versions
The only supported versions of the bot (i.e., actively developed and fixed) are:
- the one currently running on the official Discord server (updates take time to organize)
- the one from the latest commit on [`main`](https://github.com/pandaroux007/RepulsBot/tree/main) (i.e., the one running on the development server)

## Reporting a Vulnerability
You are encouraged to report any vulnerabilities (potential or confirmed). However, if the vulnerability is serious, meaning more than a simple typo in a message (in which case we ask you to create an issue), contact one of the main contributors (listed [here](https://github.com/pandaroux007/RepulsBot/tree/main?tab=readme-ov-file#main-contributors)) or me ([@pandaroux007](https://github.com/pandaroux007)) privately via Discord to avoid making the vulnerability public and to ensure it is fixed as quickly as possible.
